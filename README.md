🚀 AI Code Generator for Students

An AI-powered web application that automatically generates programming code based on user input.
This project helps students quickly generate solutions for programming problems using Artificial Intelligence.

📌 Project Description

The AI Code Generator allows users to enter a programming problem, and the system generates the corresponding code instantly using an AI model. It is designed to assist beginners and students in learning programming concepts and understanding code structure.

The application uses a Flask backend, a simple HTML/CSS frontend, and integrates AI models from Hugging Face for code generation.
